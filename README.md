# 🍽️ Feature Engineering on Restaurant Dataset

## 📖 Introduction

Feature Engineering is a crucial step in the data preprocessing pipeline. In this project, I performed feature extraction and categorical encoding on a restaurant dataset to create meaningful variables that can improve data analysis and machine learning model performance.

## 🎯 Objectives

* Extract useful information from existing columns.
* Create new features from restaurant names and addresses.
* Transform categorical variables into numerical representations.
* Prepare the dataset for future predictive modeling tasks.

## 🔍 Feature Engineering Tasks Performed

### 1. Text-Based Feature Extraction

* Calculated the length of each restaurant name.
* Calculated the length of each restaurant address.

### 2. Cuisine-Based Feature Creation

* Counted the number of cuisines offered by each restaurant.

### 3. Categorical Variable Encoding

Converted categorical columns into binary numerical values:

| Feature             | Encoding        |
| ------------------- | --------------- |
| Has Table Booking   | Yes = 1, No = 0 |
| Has Online Delivery | Yes = 1, No = 0 |
| Is Delivering Now   | Yes = 1, No = 0 |
